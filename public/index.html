Github Feed

RSS for your Github

Github Feed generates RSS feeds for your Github commits so that others can subscribe to them through an RSS aggregator like Google Reader. This might be useful for new updates to a Github Pages blog, new versioned releases of your software, or anything else you'd like to share with the world.

How it Works

Go to your repository's admin console and add this address as a post-receive hook. When you want a commit to be added to the RSS feed, add the word @rss anywhere in the commit message. The commit will then be added into our database. You can then subscribe to the RSS feed at http://ghfeed.heroku.com/username/repo. Here is an example.
